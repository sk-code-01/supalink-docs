# What is Supalink?

Supalink is instant static web hosting. Publish any file or folder and get a live URL in seconds.

```bash
./publish.sh ./dist
# => https://bright-canvas-a7k2.supalink.dev/
```

Every site gets a unique slug and is served from a global CDN via Cloudflare.

## How It Works

Publishing is a 3-step process:

<Steps>
  <Step title="Create">
    Send your file manifest to the API. You get back upload URLs for each file.
  </Step>
  <Step title="Upload">
    Upload each file to its presigned URL. Files go directly to Cloudflare R2 storage.
  </Step>
  <Step title="Finalize">
    Tell the API you're done. Your site goes live instantly at `slug.supalink.dev`.
  </Step>
</Steps>

The `publish.sh` script handles all three steps automatically.
