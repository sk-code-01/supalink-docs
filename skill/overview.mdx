## What is the Supalink Skill?

The Supalink skill lets AI coding agents (like Claude Code) publish files and folders to live URLs automatically. Once installed, just say "publish this" and the agent handles everything.

## How It Works

1. **Agent detects intent** — You say "publish this", "deploy my site", "give me a URL", or similar
2. **Skill activates** — The agent loads the Supalink skill instructions
3. **Files are published** — The agent runs `publish.sh` to scan, upload, and finalize
4. **URL returned** — You get a live URL at `slug.supalink.dev`

## Supported Agents

The skill follows the open [Agent Skills](https://agentskills.io) standard and works with:

- **Claude Code** — Full support via `npx skills add`
- **Cursor** — Via the skills system
- **Windsurf** — Via the skills system
- **Other agents** — Any agent supporting the Agent Skills standard
